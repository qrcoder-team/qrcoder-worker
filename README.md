# QRCode Worker API

This is the API implementation for [QRCoder](https://github.com/qrcoder-team/qrcoder-frontend). It uses **itty-router**, **Cloudflare Worker** and **Cloudflare Worker KV** to store QrCodes and its configurations

## Prerequisite

- Node
- npm
- Wrangler

## Setup

```bash
$ npm install --no-save

```

## Publish

```
$ wrangler login
$ wrangler publish
```
